## 📖 Sobre o Projeto

O **Imúnica** é uma plataforma educacional e analítica desenvolvida para auxiliar crianças e adolescentes a identificar sinais de desinformação digital e fake news.

O projeto foi criado durante o **Hackanave**, hackathon do projeto **Nave à Vela**, com o objetivo de unir tecnologia, educação e pensamento crítico em uma experiência acessível e interativa.

A plataforma busca ensinar usuários a analisar informações presentes na internet, compreender critérios de confiabilidade e desenvolver senso crítico diante de conteúdos digitais.

---

## 🎯 Objetivo

O principal objetivo da Imúnica é capacitar jovens usuários a:

- Identificar possíveis fake news
- Reconhecer sinais de manipulação digital
- Desenvolver pensamento crítico
- Aprender práticas de verificação de informações
- Navegar pela internet com mais segurança e consciência

---

# ✨ Funcionalidades

## 🔎 Analisador de Textos

A plataforma possui um sistema de análise textual que avalia conteúdos com base em critérios fixos relacionados à desinformação.

O analisador:
- Examina características inseridas no texto
- Identifica possíveis sinais de fake news
- Classifica o nível de desinformação
- Apresenta resultados de forma acessível e educativa

Tudo isso baseado em padrões constantemente presentes na desinformação espalhada em meio digital 

---

## 📚 Área Educativa

A área educativa reúne conteúdos voltados ao aprendizado sobre desinformação digital.

Inclui:
- Materiais de leitura
- Conteúdos educativos
- Explicações sobre fake news
- Dicas de verificação de informações
- Jogos educativos interativos

Os jogos possuem foco em ensinar, de forma prática e dinâmica, como detectar manipulações e informações falsas.

# 🚀 Como Executar o Projeto

## Clone o repositório

```bash
git clone https://github.com/PedroOliveira-N/imunica.git
```

## Entre na pasta do projeto

```bash
cd imunica
```

## Instale as dependências

```bash
npm install
```

## Execute o projeto

```bash
npm run dev
```

O projeto estará disponível em:

```bash
http://localhost:5173
```

# 🧠 Impacto Educacional e Uso da Plataforma

A Imúnica busca combater diretamente a desinformação digital por meio da educação tecnológica e do incentivo ao pensamento crítico.

A plataforma foi idealizada especialmente para jovens usuários, utilizando uma abordagem acessível, visual e interativa.

Pensando que o uso prioritário é de crianças e adolescentes, o projeto poderá ser usado em parceria com escolas e outras instituições, porém também será utilizado pelos próprios jovens, visto que o site é intuitivo e pode ser usado sem complicações.
